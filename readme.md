# AI Law Chatbot – Engineering Repository

This repository contains the **engineering structure and core logic**
for a production-grade AI Legal Chatbot built using RAG architecture.

## Features (in progress)
- Legal document ingestion (PDF, DOCX, URLs)
- Metadata extraction
- RAG-based retrieval
- Auto-update pipeline
- LLM-based legal Q&A

⚠️ Note:
All legal data, scraped content, and client-specific information
have been intentionally excluded.
